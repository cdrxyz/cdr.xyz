# cdr.xyz

Portfolio site built with [Next.js](https://nextjs.org), [Tailwind CSS](https://tailwindcss.com), and [MDX](https://mdxjs.com/).

## Getting started

This project uses [Hermit](https://cashapp.github.io/hermit/) to manage toolchain dependencies (Node.js, npm). Activate it first:

```bash
. ./bin/activate-hermit
```

Then install dependencies and start the development server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to view the website.

## Customizing

Edit files in the `/src` folder. The site will auto-update as you edit these files. Case studies live in `/src/app/work/`.
